# FastAPI Beyond CRUD (devops practice)

This is the source code for the [FastAPI Beyond CRUD](https://youtube.com/playlist?list=PLEt8Tae2spYnHy378vMlPH--87cfeh33P&si=rl-08ktaRjcm2aIQ) course. The course focuses on FastAPI development concepts that go beyond the basic CRUD operations.

For more details, visit the project's [website](https://jod35.github.io/fastapi-beyond-crud-docs/site/).

## Getting Started
Follow the instructions below to set up and run your FastAPI project.

### Prerequisites
Ensure you have the following installed:

- Docker

### Project Setup
1. Clone the project repository:
    ```bash
    git clone https://github.com/davidhsiaotw/fastapi-beyond-CRUD
    ```

2. Navigate to the project directory:
    ```bash
    cd fastapi-beyond-CRUD/
    ```

## Running the Application
run the application using Docker:
```bash
docker compose up
```
Alternatively, run the docker image directly:
```bash
docker run <image_name>
```
docker image can be found [here](https://github.com/davidhsiaotw/fastapi-beyond-CRUD/pkgs/container/fastapi-beyond-crud)

## Project Documentation
On local machine, access the documentation at:
- Swagger UI: `http://localhost:8000/api/v1/docs`
